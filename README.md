⚡ Eletro Soluções
Site institucional da empresa Eletro Soluções, especializada em manutenção e reparo de equipamentos eletrônicos e eletrodomésticos.

🚀 Tecnologias utilizadas
HTML5 → Estrutura semântica da página.

CSS3 → Estilização, responsividade e efeitos visuais.

Bootstrap Icons → Ícones modernos para redes sociais e serviços.

Google Fonts → Fonte personalizada Libertinus Mono.

JavaScript → Controle do slider/banner automático e interatividade.

🏗️ Estrutura do site
Header → Logo, menu de navegação e ícones sociais.

Banner com Slider → Imagens rotativas, setas de navegação e texto institucional.

Serviços → Cards com ícones e descrições das especialidades.

Sobre → Imagem ilustrativa, texto explicativo, localização e redes sociais.

Footer → Menu de rodapé e direitos autorais.

⚙️ Funcionamento
O slider/banner alterna automaticamente a cada 5 segundos e pode ser controlado manualmente pelas setas.

O CSS define responsividade com media queries (até 768px e 420px).

O menu usa âncoras para rolar até as seções correspondentes.

Os links sociais abrem em nova aba (target="_blank").

📱 Responsividade
Até 768px: menu e redes sociais do header somem, layout vira coluna única.

Até 420px: ícones sociais são ocultados para evitar quebra.

Texto e imagens se ajustam proporcionalmente para manter legibilidade.
